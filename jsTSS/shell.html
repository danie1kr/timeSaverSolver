Rules

  • The locomotive moves through the layout. Speed does not matter.
  • Each change in direction costs one point.
  • Each changed turnout cost one point.

HowTo

  1. Pick a track layout and number of cars.
  2. Set start and end configurations. Set your own model rail way to the same start configuration.
  3. Click [Solve] to find the quickest solution to the TimeSaver puzzle.
  4. See if you did the same solution or beat the computer!
